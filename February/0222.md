> ## 關於程式

昨天談到我們可以使用第三方套件 Radium 來操控 React 內元件的美術樣式，那有沒有不使用 Radium 的方法呢？

有的，那就是 CSS Module ，可以透過更改 Webpack 打包的 config 設定檔，讓開發者能以模組化的方式 import CSS 檔來打造想要的美術風格。
