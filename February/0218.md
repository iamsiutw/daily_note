> 關於程式

今天進入React的Section 4 - lists & conditionals:

使用條件式語法來渲染element

conditionals基本概念：

```
...

state = {
    showPersons: false
}

render() {
    let persons = null
    
    // 由showPersons的boolean值判斷persons是否有value
    if (this.state.showPersons) {
        persons = (
        <div>Amy, Bob, Carter</div>
        )
    }
    
    // JSX內使用{persons}來渲染persons
    return (
        <div>
            <p>Group: {persons}</p>
        </div>
    )
}

...
```
